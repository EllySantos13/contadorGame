# 🎮 Contador Game

Projeto desenvolvido em React Native com expo.

---

# Preview do APP

O aplicativo possui:
* contador de pontos;
* botão de aumentar;
* botão de diminuir;
* botão de resetar;
* interface estilo gamer; 

---

# Tecnologias utilizadas

* React Natives
* Expo
* JavaScript

---

# Como executar o projeto

## 1. clonar repositório
```
git clone https://github.com/EllySantos13/contadorGame.git
```

## 2. Entrar na pasta do projeto
```
cd contadorGame
```

## 3. Instalar dependências 
```
npm install
```

## 4. Rodar o projeto
```
npm start
```
